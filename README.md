# Mini-Racing-Game---Python

Overview

This fast racing game is built with Python and Pygame where you race small cars on a race track. Be sure to avoid obstacles as oil slicks hazards that spin out your car and not drive on the grass as it slows you down. This game has easy controls and 5 other AI opponents that you race with in your car. You can choose your difficults level and lap count. This game also shows you live race rankings. Along with that, it has visual effects like smoke particles, trye trails and more. The controls are told in the game. It also has a live speedometer which acts like a real one in a car.his racing game is great for people trying to see how well they can drive in a game along with AI opponents which give good compeition.


How to run it-

Install Python and Pygame (pip install pygame)

Get this file (main.py)
